这天发生了很多离奇的事情，第一就是电脑自动启动之谜：

 按道理来说，计算机睡眠状态下确实比较容易被意外唤醒，但这一般都是针对连接了其他外设导致的，比如说连接了蓝牙鼠标（2.4G接口），就很容易因为鼠标的微动导致计算机开机，但问题是，我确定当时计算机未连接可造成扰动的外设，但却仍然在早上我还在睡觉的时候自动开机了（迷惑），确切的开机时间可以从系统的任务日志中查看到自动开机时间，后来我得知实验室的服务器遭到攻击，而时间也恰好是这个时间段，但是我也确实没有说谎，不管怎么说，即使电脑开机了也必然处于断网的状态，必然无法连接到服务器，更不用说主动发起攻击，这一切都显得很魔幻，直到现在我仍无法得知当时的真相。。。而这次攻击直接导致实验室的服务器需要重新安装系统（害怕）。 我的系统详细记录也证明了这一点网络连接记录，可以看到，第一次ssh连接发生在9点多，而5点多确实有一个记录，但只是一个0kb的记录文件，无法解析当时到底发生了什么，但如果是ssh连接的话，应该不会没有记录。
